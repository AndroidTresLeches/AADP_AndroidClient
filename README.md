AADP_AndroidClient
==================

This is an Android application for Asian American Donor Program (AADP)


**Walkthrough**

Completed user stories:
- [x] Required: User can login to AADP
- [x] Required: User can view list of events sorted by date.
  * Events display event name, event date, event time, event address, coordinator name & photo.
- [x] Required: User can scroll to view more events.
- [x] Required: User can click on calender icon to save event to calendar
- [x] Required: User can click on favorite icon to save event to favorites list.
- [x] Required: User can click on the event to view the event in detail
- - [x] Required: User can click on the event to view the event in detail
- - [x] Required: User can click on the event to view the event in detail
- [x] Advanced: User can select "reply" from detail view to respond to a tweet.
